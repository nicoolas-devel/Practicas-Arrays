# Prácticas de Arrays y Bucles en JavaScript

## Descripción

Este repositorio contiene ejercicios realizados para practicar conceptos básicos e intermedios de JavaScript.

Durante estas actividades se trabajó con estructuras de control, bucles, arrays y switch , resolviendo diferentes problemas propuestos en clase.

## Temas trabajados

- Variables
- if / else
- switch
- for
- Arrays
- push()
- unshift()
- splice()
- length
- prompt()
- alert()
- console.log()

## Ejercicios realizados

1. Bucle del 1 al 20
2. Tabla de multiplicar del 5
3. Filtrar números mayores a 10
4. Estación del año
5. Switch de idiomas
6. Array de animales
7. Cálculo de IMC
